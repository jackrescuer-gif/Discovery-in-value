# Как отрисовать весь процесс Discovery в Miro или Holst

Этот гайд позволяет воссоздать **полную схему процесса Discovery** в [Miro](https://miro.com) или [Holst](https://app.holst.so) двумя способами: быстро (изображение) или правильно (нативные элементы).

---

## Способ 1 — Быстро: изображение на борде

1. **Скачай готовую диаграмму**
   - PNG: в репозитории есть `holst-process-diagram.png` (если уже экспортирован), или открой в браузере `holst-process-diagram.html` → вписать в экран (F) → скриншот/Print to PDF.
   - Либо открой `process-board.html` — интерактивная схема с зумом; экспортируй видимую область.

2. **Miro**: Новый борд → меню «+» → Image / Upload → перетащи PNG. Масштабируй под холст.

3. **Holst**: [app.holst.so](https://app.holst.so) → New board → перетащи PNG на борд или «+» → Image.

**Плюс:** быстро. **Минус:** нельзя редактировать блоки и связи.

## Miro: Bulk-режим (много карточек сразу)

В Miro: Sticky note → внизу «Bulk mode» / «Режим пакетного ввода». Вставка текста: **каждая строка = одна стикер-нота**.

Готовый список строк для копирования лежит в файле **`miro-bulk-cards.txt`** в этом репозитории. Скопируй содержимое целиком и вставь в Bulk mode — получишь все карточки процесса; останется расставить их по зонам и соединить стрелками по таблице выше.

---

## Holst: пошагово

1. **Фоны**: Rectangle → 5 зон по таблице 2.1, подписи сверху.
2. **Карточки**: Rounded rectangle для шагов, Diamond для D1–D4, Ellipse для старта и финиша. Цвета — по таблице 2.2.
3. **Стрелки**: Connector / Arrow между блоками по таблице 2.3; для «Нет/итерация» выбери пунктир и красный цвет.
4. При необходимости используй `holst-process-diagram.png` как фоновое изображение под бордом, чтобы выровнять блоки.
