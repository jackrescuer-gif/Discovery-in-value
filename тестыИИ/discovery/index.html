Value Creation Stream

Полная визуализация трансформации инициатив в реализованную ценность: от фильтрации идей через портфель управления, выбор discovery framework в VDO, до доставки готового решения

1️⃣ Initiative Governance

Управление портфелем инициатив

  • Filtration: Отбор идей по критериям ценности
  • PI Planning: Приоритизация инициатив
  • Portfolio: Управление одобренными инициативами

2️⃣ Discovery Governance

Value Design Office управляет открытиями

  • Framework Selection: VDO выбирает из 5 типов
  • Discovery Execution: Специфичный для framework процесс
  • Delivery Package: Готовый пакет для Delivery

3️⃣ Delivery Execution

Реализация и доставка ценности

  • GO/NO-GO Review: Governance gate принятия решения
  • Delivery Routing: Направление в нужный канал
  • Implementation: Разработка и реализация

🔑 Ключевые концепции новой модели

📊 Portfolio Management

После Filtration и PI Planning инициативы входят в управляемый портфель, где фиксируется их статус, резервируются ресурсы и назначаются спонсоры. Это обеспечивает видимость и контроль над всеми текущими инициативами.

🎯 Value Design Office

VDO — это центральная функция управления discovery, которая классифицирует каждую инициативу, выбирает правильный framework (из 5 типов) и координирует специфичный для этого типа процесс открытий.

🔄 5 Discovery Frameworks

Product Discovery (новые фичи, клиентская ценность) | Project Initiation (чёткие требования, infra) | Operational Improvement (процессы, затраты) | Strategic Exploration (новые рынки) | Experimentation (быстрое обучение)

📦 Delivery Package

Стандартный выход Discovery независимо от framework. Содержит: Problem Statement, Solution Concept, Scope, Implementation Approach, Roadmap, Team Model, KPIs, Budget. GO/NO-GO Review решает передавать ли в Delivery.
