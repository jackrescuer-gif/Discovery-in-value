Перенос схемы в Holst вручную

Способ 1 — Быстро: импортировать как изображение

Шаг 1
Сделай скриншот или скачай PNG
Нажми F чтобы вписать схему в экран → Ctrl+Shift+S (или расширение Full Page Screenshot). Либо возьми файл из репозитория.
Шаг 2
Открой Holst → создай борд
Перейди на app.holst.so → New board → пустой шаблон.
Шаг 3
Вставь изображение
Перетащи PNG прямо на борд или нажми + → Image → загрузи файл. Масштабируй на весь борд.
Файл PNG: тестыИИ/discovery/cross-process.png (после экспорта). Перетащи в браузер → Ctrl+P → Save as PDF → конвертируй.

Способ 2 — Правильно: нативные элементы Holst

Шаг 1
Создай фоновые зоны фаз
Rectangle (без рамки, заливка 20% прозрачность) под каждую из 5 зон. Подпиши название фазы сверху.
Шаг 2
Добавь карточки шагов
Rounded Rectangle для каждого блока. Двойной клик — текст. 3-4 строки с описанием.
Шаг 3
Ромбы-решения
Diamond/Rhombus для «Выбор фреймворка». Заливка #fff9db, рамка #f59f00.
Шаг 4
Соедини стрелками
Connector → Arrow. Основной поток — серый/синий. Пивот/обратная связь — красный пунктир.
