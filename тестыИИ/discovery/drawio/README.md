# Discovery diagrams in draw.io format

Ниже — исходники диаграмм в формате `.drawio` для корпоративного редактирования в diagrams.net / draw.io.

## Файлы

1. `00-macro-3-phases.drawio` — макро-схема 3 фаз Discovery  
2. `01-phase-1-research.drawio` — фаза 1 (исследование и приоритизация)  
3. `02-phase-2-1-validation.drawio` — фаза 2.1 (валидация гипотез)  
4. `03-phase-2-2-prototyping.drawio` — фаза 2.2 (прототипирование)  
5. `04-validation-detailed.drawio` — детальная схема валидации  
6. `05-prototyping-detailed.drawio` — детальная схема прототипирования  
7. `06-phase-3-delivery.drawio` — фаза 3 (готовность и handoff в Delivery)  
8. `07-roles-interaction.drawio` — роли и взаимодействие команды  

## Как открыть

- В draw.io / diagrams.net: **File → Open From → Device**
- Выберите нужный `.drawio` файл из этой папки.
