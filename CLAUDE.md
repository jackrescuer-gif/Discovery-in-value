# Claude Code — Правила для этого репозитория

## Разрешённые операции (без запроса подтверждения)

- Редактировать любые файлы в `тестыИИ/discovery/`
- Коммитить изменения с осмысленным сообщением
- Пушить в ветки с префиксом `claude/`
- Делать `git push -u origin <ветка>` для feature-веток
- Создавать новые файлы в `тестыИИ/discovery/` (html, css, js)
- Читать любые файлы репозитория

## Деплой

- **Разрешено**: push в `claude/*` ветки без подтверждения
- **Разрешено**: merge feature-ветки в `main` через PR (gh pr merge)
- **Требует подтверждения**: прямой push в `main` или `master`
- **Запрещено**: force push в `main`/`master`

## Рабочий процесс

1. Разработка ведётся на ветках `claude/<описание>-<ID сессии>`
2. Коммиты — по завершению логической единицы работы
3. Push выполняется командой `git push -u origin <ветка>`
4. PR создаётся при необходимости через `gh pr create`
